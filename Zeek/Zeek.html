¿Qué es Zeek?

Zeek (anteriormente Bro) es un sistema de monitoreo de seguridad de red de código abierto que proporciona análisis profundo de tráfico, detección de intrusiones y análisis forense en tiempo real.

Zeek interpreta el tráfico de red como una serie de eventos y aplica scripts de política para analizar el comportamiento, generando registros detallados y alertas de seguridad personalizables.

# Script básico de detección en Zeek
module ZeekExample;

event connection_established(c: connection) {
  if (c$id$resp_h == 192.168.1.1) {
    print("Conexión establecida con servidor crítico");
  }
}

event http_request(c: connection, method: string, original_URI: string) {
  if (method == "POST" && "admin" in original_URI) {
    NOTICE([$note=HTTP::Admin_Access,
      $msg="Acceso administrativo detectado",
      $conn=c]);
  }
}

Scripts de Detección de Zeek

Scripts principales:

  • Scan Detection - Detección de escaneos de puertos y hosts
  • Malware Detection - Identificación de tráfico malicioso
  • Policy Violations - Detección de violaciones de políticas
  • Anomaly Detection - Detección de comportamiento anómalo
  • Threat Intelligence - Integración con feeds de amenazas
  • Protocol Analysis - Análisis profundo de protocolos

Ejemplo de script de detección de escaneo:

# Detección de escaneo de puertos en Zeek
module ScanDetection;

global scan_threshold: count = 25 &redef;

event connection_attempt(c: connection) {
  local scanner = c$id$orig_h;
  local target = c$id$resp_h;

  if (++c$num_attempts > scan_threshold) {
    NOTICE([$note=Scan::Address_Scan,
      $msg=fmt("Escaneo detectado desde %s hacia %s", scanner, target)",
      $src=scanner, $dst=target]);
  }
}

# Este script alerta cuando un host intenta conectar a más de 25 hosts diferentes
