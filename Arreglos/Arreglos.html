Arreglos en JavaScript

Una demostración elegante de métodos y operaciones con arreglos. Este ejemplo muestra cómo manipular, transformar y analizar datos utilizando las funcionalidades modernas de JavaScript.

Métodos Básicos

const frutas = ['Manzana', 'Banana', 'Cereza'];
frutas.push('Durazno'); // Agrega al final
frutas.pop(); // Elimina el último
frutas.shift(); // Elimina el primero
frutas.unshift('Uva'); // Agrega al inicio

Transformación (map, filter, reduce)

const numeros = [1, 2, 3, 4, 5];

// map: Transforma cada elemento
const cuadrados = numeros.map(n => n * n);

// filter: Filtra elementos
const pares = numeros.filter(n => n % 2 === 0);

// reduce: Reduce a un solo valor
const suma = numeros.reduce((acc, n) => acc + n, 0);

Métodos Modernos

const datos = [42, 18, 73, 29, 56];

// find: Encuentra el primer elemento que cumple la condición
const mayor50 = datos.find(x => x > 50);

// some: Verifica si al menos uno cumple
const hayPares = datos.some(x => x % 2 === 0);

// every: Verifica si todos cumplen
const todosPositivos = datos.every(x => x > 0);

// includes: Verifica si contiene un valor
const tiene42 = datos.includes(42);

Desestructuración y Spread Operator

// Desestructuración de arreglos
const colores = ['Rojo', 'Verde', 'Azul', 'Amarillo'];
const [primero, segundo, ...resto] = colores;

// Spread operator para combinar arreglos
const masColores = ['Morado', 'Naranja'];
const todosColores = [...colores, ...masColores];

// Spread para copiar un arreglo
const copiaColores = [...colores];
