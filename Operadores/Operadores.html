Operadores JavaScript

Una representación visual y estética de los diferentes tipos de operadores en JavaScript, con ejemplos interactivos que puedes ejecutar.

Operadores Aritméticos

+ - * / % **

Realizan operaciones matemáticas básicas entre valores numéricos. Incluyen suma, resta, multiplicación, división, módulo y exponenciación.

Ejemplo:
let a = 10;
let b = 3;
let suma = a + b; // 13
let modulo = a % b; // 1

Operadores de Asignación

= += -= *= /=

Asignan valores a variables. Pueden combinarse con operadores aritméticos para realizar operaciones y asignaciones simultáneas.

Ejemplo:
let x = 5;
x += 3; // x = 8
x *= 2; // x = 16
x /= 4; // x = 4

Operadores de Comparación

== === != !== > <

Comparan dos valores y devuelven un valor booleano (true o false). "===" y "!==" comparan valor y tipo, mientras que "==" y "!=" solo comparan valor.

Ejemplo:
5 == '5' // true
5 === '5' // false
10 > 5 // true

Operadores Lógicos

&& || !

Combinan expresiones booleanas. "&&" (AND) devuelve true si ambas expresiones son verdaderas, "||" (OR) devuelve true si al menos una es verdadera, y "!" (NOT) invierte el valor.

Ejemplo:
let esMayor = (10 > 5) && (3 < 7); // true
let esVerdadero = !(5 === '5'); // true

Incremento/Decremento

++ --

Aumentan o disminuyen el valor de una variable en 1. Pueden usarse como prefijo (++x) o sufijo (x++), lo que afecta al valor devuelto por la expresión.

Ejemplo:
let contador = 5;
contador++; // contador = 6
++contador; // contador = 7
contador--; // contador = 6

Operador Ternario

? :

Es una forma abreviada de escribir una estructura if-else. Evalúa una condición y devuelve un valor si es verdadera, o otro si es falsa.

Ejemplo:
let edad = 18;
let puedeVotar = (edad >= 18) ? "Sí puede" : "No puede";
// "Sí puede"
